About Claude Tower Defense

Claude Tower Defense is a lightweight browser-based tower defense game, designed to run seamlessly without any installation or plugins. It was built as an experimental project to explore game development using modern front-end technologies.

Tech Stack

  • Javacript

  • HTML

  • CSS

Open Source

This project is completely open source. You're welcome to explore, fork, or contribute to its development on my Github

Purpose

Beyond entertainment, this project serves as a proof of concept for building 2D games with React and Canvas. It's also meant as a learning resource for anyone interested in experimenting with front-end development and indie game design.
